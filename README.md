## Exercício 3 - Medir tempos com escalonamentos RMPO e RMPO Inverso
Compilar exercício 3 com escalonamento RMPO
```bash
make ex3_RMPO
```
Correr
```bash
sudo ./ex3_RMPO
```

Compilar exercício 3 com escalonamento RMPO Inverso
```bash
make ex3_InvRMPO
```
Correr
```bash
sudo ./ex3_InvRMPO
```
